
# Trader Behavior Analysis under Market Sentiment

This project analyzes how trader behavior and profitability differ under Fear and Greed
market conditions using the Bitcoin Fear & Greed Index and Hyperliquid historical trading data.

## Project Structure
- notebook_1.ipynb : Google Colab notebook containing the full analysis
- csv_files/       : Raw and processed datasets
- outputs/         : Visualizations generated during analysis
- ds_report.pdf    : Summary report with insights and conclusions

## Key Insights
- Trading activity and volume are higher during Fear periods
- Average daily profitability is higher during Fear
- Fear periods show higher PnL volatility, indicating higher risk
